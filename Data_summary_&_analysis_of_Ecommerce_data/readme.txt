The main aim of the project is to clean, summarize and analyze an Ecommerce dataset containing data of over 100 products purchased from moree than 25 different countries for the year 2011. Data was imported using RPostGRESQL in RStudio and cleaned, summarized using PostgreSQL in DBeaver Database Manager. Specific objectives of the project are to 1) Clean, explore and summarize the Ecommerce dataset, 2) Calculate revenue based Key Performance Indices of the dataset, 3) Perform the Recency, Frequency, Monetary analysis with customer segmentation and 4 ) Data Visualization of the analyses. The first three objectives will be fulfilled using PostgreSQL while the fourth objective will be done using Tableau Public.
